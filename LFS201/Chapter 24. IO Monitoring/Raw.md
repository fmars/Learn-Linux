# iostat
- iostat shows CPU statistics and IO statistics for devices
- CPU
  - user
  - nice 
  - iowait
  - steal
  - idle
  
- Devices (iostat -k)
  - tps: transfer per second
  - kB_read/s
  - kB_write/s
  - kB_read
  - kB_write
  
- `iostat -xk`: extended mode
